Edit Existing Deployment

Add or remove accounts from an existing MAP auto-tagger deployment

This tool modifies an existing deployment created by the MAP Auto-Tagger Configurator. It generates an update.sh script that updates the account scope without redeploying.

Action

Choose whether to add or remove accounts from the MAP scope

Add accounts to MAP scope

New accounts will start being tagged with the MAP tag. Backfill runs automatically for historical resources.

Remove accounts from MAP scope

Accounts will stop being tagged. Existing tags are not removed. The Lambda remains deployed but inactive.
