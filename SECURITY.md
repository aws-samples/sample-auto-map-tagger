# Security Policy

## Reporting Security Issues

If you discover a potential security issue in this project, please do **not** open a public GitHub issue.

Instead, report it through AWS's vulnerability reporting page:
**http://aws.amazon.com/security/vulnerability-reporting/**

We take security issues seriously and will work with you to address the issue promptly.

## Supported Versions

Security updates are provided for the latest version of this project.

## What to Include in a Report

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations (if known)
